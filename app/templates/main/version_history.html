Version 1.5.1

March 2026

Literature Review — PubMed Search

  • • LitReview Module: New blueprint at /litreview — PubMed-powered literature review integrated into PanelMerge
  • • PubMed Gene Search: Search PubMed by gene name or free-text term via the NCBI Entrez API (Biopython); configurable result count
  • • Search Results Page: Ranked article cards showing title, authors, journal, year, abstract preview, PMID badge, and DOI links
  • • Article Detail View: Full abstract, complete author list, MeSH terms, publication types, and external PubMed/DOI links
  • • Search History: Paginated history of all your past searches with result counts and quick re-open links
  • • Two-Layer Caching: Redis cache (1-hour TTL) for search PMID lists; database cache (7-day rolling TTL) for article metadata — minimises repeat API calls
  • • Search Progress Overlay: Spinner shown during long searches with a Cancel button (uses AbortController)
  • • Excel Export: Download search results as a formatted .xlsx with styled header row, column widths, and text-wrapped abstract column
  • • CSV Export: Download search results as a UTF-8 CSV (Rank, PMID, Title, Authors, Journal, Year, DOI, Abstract, MeSH Terms)
  • • Swagger/OpenAPI Documentation: Flask-RESTX API docs at /api/v1/docs covering search, results, and article detail endpoints
  • • Per-Article View Tracking: UserArticleAction records view counts and last-viewed timestamps per user
  • • Database Schema: New tables — literature_articles, literature_searches, search_results, user_article_actions
  • • New Dependencies: biopython, xmltodict, flask-restx

Version 1.5.0

March 2026

Saved Panel Library & Security Enhanced

  • • Saved Panel Library System: Complete personal panel storage with modification tracking and version control
  • • Version Control: Git-like versioning with configurable retention, tagging, and branch visualization
  • • My Panels Profile Tab: Comprehensive panel management interface with sortable grid and advanced filtering
  • • Multi-Format Export: Export panels in Excel, CSV, TSV, and JSON formats with metadata and version history
  • • Export Wizard: Intuitive export interface with custom filenames, column selection, and template creation
  • • Google Cloud Storage Integration: Multi-backend storage architecture with primary GCS and local backup
  • • Advanced Password Security: Password history tracking, reuse prevention, and strong password enforcement
  • • Account Lockout Protection: Automatic lockout after multiple failed attempts with admin unlock interface
  • • Password Reset Security: Single-use tokens, suspicious activity detection, and geographic anomaly analysis
  • • Admin Password Override: Secure admin password reset with temporary passwords and session termination
  • • Email Change Verification: Verification required for email changes with dual email notifications
  • • Enhanced Audit Log Viewer: Improved security audit log interface with advanced filtering capabilities
  • • Advanced Filtering: Multi-criteria panel filtering by status, version, date, and gene count
  • • API Enhancements: 15 new saved panel API endpoints with enhanced rate limiting
  • • Database Testing Suite: Comprehensive 50+ test suite for schema, integrity, and security validation
  • • Panel Sharing: Share saved panels with other users and manage permissions
  • • Batch Operations: Export multiple panels simultaneously with template support
  • • Storage Abstraction: Pluggable storage backend architecture for deployment flexibility

Database: PostgreSQL (production), SQLite (local), Redis (caching/sessions)

Deployment: Google Cloud Platform with Cloud SQL and Cloud Storage

Testing: pytest, unittest, comprehensive database and API testing
